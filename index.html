PowerPoint 접근 권한 필요

슬라이드를 원본 그대로 가져오려면 이 앱이 Microsoft PowerPoint를 제어할 수 있도록 macOS 자동화(Automation) 권한을 허용해야 합니다.

1. 시스템 설정 앱을 엽니다
2. 개인 정보 보호 및 보안 → 자동화 로 이동합니다
3. 이 앱(천명 프로그램) 아래에서 Microsoft PowerPoint 토글을 켭니다
4. 아래 다시 시도 버튼을 누릅니다
