## 1. 설치하기

### 1-1. 설치 파일 내려받기

GitHub 릴리스 페이지(https://github.com/dbrdid/cheonmyeong/releases)에서 최신 버전의 `.exe` 파일을 내려받습니다.

- **설치 마법사 버전** `천명프로그램-x.x.x-Setup.exe` — 시작 메뉴에 등록됩니다.
- **포터블 버전** `천명프로그램-x.x.x-portable.exe` — 설치 없이 바로 실행됩니다.

![릴리스 페이지에서 파일 내려받기](docs/screenshot-01-download.png)

---

### 1-2. Windows SmartScreen 경고 우회

코드 서명이 적용되지 않은 프로그램은 처음 실행 시 아래와 같은 파란색 경고창이 표시될 수 있습니다.  
**정상입니다. 아래 순서대로 진행하세요.**

![SmartScreen 경고창](docs/screenshot-02-smartscreen-warning.png)

**① "추가 정보"를 클릭합니다.**

![추가 정보 클릭](docs/screenshot-03-smartscreen-moreinfo.png)

**② "실행" 버튼을 클릭합니다.**

![실행 버튼 클릭](docs/screenshot-04-smartscreen-run.png)

> 이 경고는 최초 1회만 표시됩니다. 이후 실행 시에는 나타나지 않습니다.

---

### 1-3. 설치 마법사 진행 (Setup 버전)

포터블 버전은 이 단계를 건너뛰어도 됩니다.

![설치 마법사 시작](docs/screenshot-05-installer-start.png)

1. 설치 위치를 선택합니다 (기본값 권장).
2. "설치" 버튼을 클릭합니다.
3. 설치가 완료되면 바탕화면과 시작 메뉴에 아이콘이 생성됩니다.

![설치 완료](docs/screenshot-06-installer-done.png)

---

## 2. 처음 실행하기

설치 후 바탕화면의 **천명 프로그램** 아이콘을 더블클릭하거나, 포터블 버전은 `.exe` 파일을 더블클릭합니다.

![메인 화면](docs/screenshot-07-main-window.png)

**처음 실행 시 자동으로:**
- 두 번째 모니터(프레젠테이션 출력 창)가 열립니다.
- 모니터가 한 대뿐이면 출력 창이 같은 화면 안에 열립니다.

> **두 번째 모니터가 보이지 않을 때:** 상단 메뉴 → 보기 → 출력 화면 열기를 클릭하세요.

## 3. PPTX 파일 가져오기

### 3-1. 파일 선택

상단 메뉴 또는 화면의 **"PPT 가져오기"** 버튼을 클릭합니다.

![PPT 가져오기 버튼](docs/screenshot-08-import-button.png)

파일 선택 창이 열리면 `.pptx` 파일을 선택하고 **열기**를 클릭합니다.

![파일 선택 창](docs/screenshot-09-file-picker.png)

---

### 3-2. 변환 진행

파일을 불러오는 동안 진행 표시줄이 표시됩니다.  
슬라이드 수에 따라 수 초에서 1분 정도 소요될 수 있습니다.

![변환 중 화면](docs/screenshot-10-converting.png)

> **처음 가져올 때 PowerPoint가 잠깐 열렸다 닫힐 수 있습니다.** 정상 동작입니다. 건드리지 마세요.

---

### 3-3. 슬라이드 목록 확인

변환이 완료되면 왼쪽 패널에 슬라이드 목록이 표시됩니다.

![슬라이드 목록](docs/screenshot-11-slide-list.png)

## 4. 슬라이드 표시하기

슬라이드 목록에서 원하는 슬라이드를 클릭하면 출력 화면(두 번째 모니터)에 즉시 표시됩니다.

![슬라이드 출력 화면](docs/screenshot-12-output-screen.png)

### 주요 기능

| 기능 | 방법 |
|------|------|
| 다음 슬라이드 | 키보드 → 또는 Page Down |
| 이전 슬라이드 | 키보드 ← 또는 Page Up |
| 화면 블랙아웃 | B 키 또는 블랙 버튼 |
| 화면 화이트아웃 | W 키 또는 화이트 버튼 |
| 전체화면 전환 | F11 또는 메뉴 → 보기 → 전체화면 |

## 5. LibreOffice 설치 안내 (PowerPoint가 없는 경우)

Microsoft PowerPoint가 설치되어 있지 않으면 **LibreOffice**를 설치해야 PPTX 변환이 정상 동작합니다.

### 5-1. LibreOffice 내려받기

1. 아래 주소를 브라우저에 입력합니다.  
   `https://www.libreoffice.org/download/libreoffice-fresh/`

2. **Windows 64-bit** 버전을 내려받습니다.

![LibreOffice 다운로드 페이지](docs/screenshot-13-libreoffice-download.png)

### 5-2. 설치

1. 내려받은 `.msi` 파일을 더블클릭합니다.
2. 기본 설정 그대로 "다음" → "설치"를 클릭합니다.
3. 설치 완료 후 **천명 프로그램을 재시작**합니다.

> LibreOffice는 별도로 실행하지 않아도 됩니다. 천명 프로그램이 내부적으로 자동으로 사용합니다.

## 6. 한글 폰트가 깨질 때

슬라이드의 한글 텍스트가 □□□□ 또는 이상한 문자로 표시되면 폰트가 설치되지 않은 것입니다.

### 6-1. 원인 확인

PPTX 파일을 만들 때 사용한 폰트(예: 나눔고딕, 고도체, 배달의민족 폰트 등)가 이 컴퓨터에 없을 때 발생합니다.

### 6-2. 해결 방법

**방법 A: 해당 폰트 설치하기 (권장)**

1. 파일을 만든 컴퓨터에서 어떤 폰트를 사용했는지 확인합니다.
2. 해당 폰트를 인터넷에서 내려받아 설치합니다.
   - 나눔 폰트: https://hangeul.naver.com/font
   - 구글 폰트: https://fonts.google.com/?subset=korean
3. 폰트 파일(`.ttf` 또는 `.otf`)을 더블클릭 → **설치** 클릭.
4. 천명 프로그램을 재시작합니다.

![폰트 설치 화면](docs/screenshot-14-font-install.png)

**방법 B: PPTX 파일을 맑은 고딕으로 변환하기**

PowerPoint 또는 LibreOffice Impress에서 파일을 열고, 모든 텍스트 폰트를 **맑은 고딕** (Windows 기본 폰트)으로 변경한 뒤 다시 저장합니다.

## 7. 자주 묻는 질문 FAQ

---

### ❓ PPT가 안 열려요 / 변환이 실패해요

**원인 1 — PowerPoint도 LibreOffice도 설치되지 않음**

천명 프로그램은 PPTX 변환을 위해 PowerPoint 또는 LibreOffice가 필요합니다.  
→ [5. LibreOffice 설치 안내](#5-libreoffice-설치-안내-powerpoint가-없는-경우) 를 참고하여 LibreOffice를 설치하세요.

**원인 2 — 파일 경로에 특수문자 포함**

파일 경로나 파일명에 `&`, `#`, `%` 등 특수문자가 있으면 변환에 실패할 수 있습니다.  
→ 파일을 `C:\PPT\` 처럼 단순한 경로로 옮긴 후 다시 시도하세요.

**원인 3 — 파일이 열려 있음**

PowerPoint나 다른 프로그램에서 해당 PPTX 파일을 열어 둔 상태면 변환에 실패합니다.  
→ 다른 프로그램을 모두 닫고 다시 시도하세요.

---

### ❓ 한글이 깨져요

→ [6. 한글 폰트가 깨질 때](#6-한글-폰트가-깨질-때) 를 참고하세요.

---

### ❓ 프로그램이 느려요 / 변환 시간이 너무 오래 걸려요

**슬라이드 수에 따라 시간이 다릅니다.**

| 슬라이드 수 | 예상 소요 시간 (PowerPoint) | 예상 소요 시간 (LibreOffice) |
|------------|---------------------------|---------------------------|
| 10장 이하 | 10~20초 | 20~40초 |
| 30장 | 30~60초 | 1~2분 |
| 50장 이상 | 1~3분 | 3~5분 |

첫 변환 이후에는 이미지가 캐시되어 빠르게 불러옵니다.

**느릴 때 시도할 것:**
- 다른 프로그램을 닫아 메모리를 확보합니다.
- PPTX 파일 내 고해상도 이미지가 많으면 시간이 오래 걸립니다. 불필요한 이미지를 줄이세요.
- LibreOffice 대신 PowerPoint를 사용하면 더 빠릅니다.

---

### ❓ 두 번째 모니터에 슬라이드가 안 나타나요

1. 모니터 연결 케이블이 제대로 꽂혀 있는지 확인합니다.
2. Windows 설정 → 디스플레이 → 여러 디스플레이 → **"확장"** 으로 설정합니다 (미러링 아님).
3. 천명 프로그램 상단 메뉴 → 보기 → **출력 화면 열기**를 클릭합니다.
4. 출력 창을 두 번째 모니터로 드래그합니다.

---

### ❓ 업데이트 알림이 떴어요

프로그램 실행 시 새 버전이 감지되면 화면 하단에 알림이 표시됩니다.  
프로그램을 종료하면 자동으로 업데이트가 설치됩니다.  
지금 업데이트하고 싶지 않다면 알림을 닫으면 됩니다.

---

### ❓ SmartScreen 경고가 계속 나타나요

처음 한 번만 표시되는 것이 정상입니다. 두 번 이상 반복되면:
1. 내려받은 `.exe` 파일을 삭제합니다.
2. 공식 GitHub 릴리스 페이지(https://github.com/dbrdid/cheonmyeong/releases)에서 다시 내려받습니다.
3. 다른 출처(카카오톡, 이메일 첨부 등)에서 받은 파일은 사용하지 마세요.
