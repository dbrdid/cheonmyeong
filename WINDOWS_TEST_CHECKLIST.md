## SmartScreen 경고 대응 가이드

1. `Windows의 PC 보호` 팝업 → **"추가 정보"** 클릭
2. **"실행"** 버튼 클릭
3. 경고 문구 및 화면 스크린샷 저장 (배포 시 사용자 안내 문서에 첨부)

> **참고:** 코드 서명(Code Signing Certificate) 적용 시 SmartScreen 경고 없어짐.  
> 현재 portable 빌드는 서명 없음 → 경고 예상됨.
